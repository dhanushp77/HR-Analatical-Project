# 📊 HR Analytics Dashboard

An interactive **Power BI HR Analytics Dashboard** that provides insights into employee attrition, salary trends, work-life balance, promotions, and department performance.

## 🚀 Features
- Employee Overview
- Attrition Analysis
- Monthly Income Analysis
- Department-wise Insights
- Work-Life Balance Analysis
- Interactive Filters (Department, Gender, Job Role, Education, Marital Status)

## 🛠 Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Excel

## 📌 Key Insights
- Analyze employee attrition trends.
- Compare attrition across departments.
- Study salary vs attrition relationship.
- Evaluate work-life balance by job role.
- Monitor average working years by department.
